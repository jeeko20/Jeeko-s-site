Commandes CMD & PowerShell

Les commandes indispensables pour administrer, dépanner et automatiser votre système Windows.

Commandes système de base

ipconfig /all - Affiche la configuration IP détaillée de votre ordinateur

ipconfig /all

systeminfo - Affiche des informations détaillées sur le système d'exploitation, le matériel et les correctifs

systeminfo

tasklist | findstr "chrome" - Liste les processus Chrome en cours

tasklist | findstr "chrome"

shutdown /s /t 0 - Éteint immédiatement l'ordinateur

shutdown /s /t 0

Commandes de gestion des fichiers

dir /w - Liste le contenu d'un répertoire en format large

dir /w

xcopy source destination /E /H /C /I - Copie un dossier récursivement avec tous les fichiers cachés

xcopy C:\source D:\backup /E /H /C /I

del /S /Q *.tmp - Supprime tous les fichiers .tmp dans le dossier et sous-dossiers sans confirmation

del /S /Q *.tmp

tree /F - Affiche l'arborescence des dossiers et fichiers

tree /F

Commandes réseau avancées

ping -t google.com - Ping continu jusqu'à interruption (Ctrl+C)

ping -t google.com

tracert google.com - Trace le chemin réseau jusqu'à la destination

tracert google.com

netstat -ano - Affiche toutes les connexions actives avec les PID

netstat -ano

nslookup google.com - Interroge les serveurs DNS pour obtenir l'adresse IP

nslookup google.com

Commandes PowerShell utiles

Get-Process | Sort-Object CPU -Descending | Select-Object -First 5 - Top 5 processus gourmands en CPU

Get-Process | Sort-Object CPU -Descending | Select-Object -First 5

Get-NetIPAddress - Affiche les adresses IP réseau (équivalent moderne d'ipconfig)

Get-NetIPAddress

Get-Service | Where-Object {$_.Status -eq "Running"} - Liste les services actifs

Get-Service | Where-Object {$_.Status -eq "Running"}

Installation de Windows 10/11

Guide pas à pas avec captures d'écran, conseils d'experts et solutions aux erreurs courantes.

Préparation avant l'installation

1

Sauvegardez vos données

Utilisez un disque dur externe ou OneDrive pour sauvegarder documents, photos, musiques et paramètres. Exportez aussi vos mots de passe navigateur.

2

Téléchargez l'outil de création de média

Rendez-vous sur le site officiel Microsoft, téléchargez "Media Creation Tool" et créez une clé USB bootable (min. 8 Go).

3

Vérifiez la compatibilité (surtout pour Windows 11)

Utilisez "PC Health Check" pour vérifier le TPM 2.0, Secure Boot, CPU compatible et 4 Go de RAM minimum.

Processus d'installation

1

Boot sur la clé USB

Redémarrez, appuyez sur F2/F12/DEL pour entrer dans le BIOS/UEFI. Mettez la clé USB en premier dans l'ordre de boot. Sauvegardez et quittez.

2

Configuration initiale

Choisissez langue, clavier, puis "Installer maintenant". Entrez votre clé de produit (optionnel) ou cliquez sur "Je n'ai pas de clé produit".

3

Partitionnement du disque

Sélectionnez le disque, supprimez les partitions existantes (si réinstall), puis cliquez sur "Nouveau" > "Appliquer". Windows crée automatiquement les partitions nécessaires.

4

Attendez la copie des fichiers

Le PC va redémarrer plusieurs fois. Ne débranchez pas la clé avant le redémarrage final. L'installation prend 20 à 45 minutes.

Configuration post-installation

1

Paramètres initiaux

Choisissez votre région, clavier, compte Microsoft (ou local), confidentialité (désactivez la télémétrie si souhaité) et mot de passe.

2

Installation des pilotes

Windows Update installe la plupart des pilotes. Pour les cartes graphiques NVIDIA/AMD, téléchargez les pilotes depuis le site officiel.

3

Optimisations essentielles

Désactivez les apps inutiles au démarrage, installez un antivirus (Windows Defender est suffisant), activez les sauvegardes et les mises à jour automatiques.
